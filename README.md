## Как запускать воркфлоу (Kilo Code Slash Commands)

В редакторе Kilo Code можно вызвать воркфлоу из каталога `.kilocode/workflows/` командой:

- `/project-setup.md` — первичная настройка окружения (Poetry или pip)
- `/submit-pr.md` — подготовка и отправка PR
- `/release.md` — релиз: SemVer, CHANGELOG, тег, zip в `versions/`
- `/code-review.md` — подготовка к ревью (линты, тесты, диагностика)
- `/security-scan.md` — поиск секретов и проблем безопасности
- `/start-ticket.md` — завести ветку под задачу
- `/docs-refresh.md` — обновить документацию и диаграммы
- `/sync-fork.md` — синхронизация форка с upstream
- `/db-migration.md` — генерация и применение миграций (Alembic)
- `/test-and-coverage.md` — запустить тесты и покрытие

> Вопросы воркфлоу к вам будут на русском, сами инструкции внутри файлов — на английском.
